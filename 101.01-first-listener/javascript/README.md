## Quick Start

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Set up environment variables**
   ```bash
   cp .env.example .env
   ```
   Edit `.env` with your Flatfile credentials.

3. **Run the listener**
   ```bash
   npx flatfile develop
   ```

The listener will start and connect to your Flatfile environment.
